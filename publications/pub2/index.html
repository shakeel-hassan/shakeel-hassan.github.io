Scalable Contrastive Learning for Multimodal Representation

Alice Chen, Shakeel Hassan, Bob Williams · ICML 2024 – International Conference on Machine Learning

📄 Paper (arXiv) 💻 Code (GitHub) 📋 BibTeX

[ Architecture diagram ]

Abstract. Contrastive learning has achieved remarkable success in unimodal settings, but scaling it to multimodal data—where images, text, and audio must be jointly aligned—remains challenging due to computational cost and data imbalance. We introduce ScaleCL, a memory-efficient contrastive learning framework that employs cross-modal momentum queues and adaptive hard-negative mining to achieve state-of-the-art zero-shot transfer on 12 benchmarks while using 3× less GPU memory than comparable methods.

Background

Contrastive objectives train encoders to push representations of semantically similar pairs together and dissimilar pairs apart. CLIP showed this principle works spectacularly for image–text pairs at scale. However, extending this to three or more modalities introduces combinatorial blowup in negative pairs and requires careful balancing across modality-specific data distributions.

ScaleCL Framework

ScaleCL introduces three key contributions:

  • Cross-modal momentum queues maintain large negatives pools without storing full batch embeddings, reducing memory by 3×.
  • Adaptive hard-negative mining dynamically adjusts negative sampling temperature per modality pair, improving training signal quality.
  • Hierarchical alignment loss aligns representations at both fine-grained and coarse-grained semantic levels.

Results

On 12 zero-shot transfer benchmarks spanning image classification, video retrieval, and audio-text matching, ScaleCL improves over CLIP by an average of 4.3 points while reducing peak GPU memory from 48 GB to 16 GB.
