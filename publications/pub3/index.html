Interactive Data Visualization with Reinforcement Learning from Human Feedback

Shakeel Hassan, Carol Lee, David Kumar, Eve Martinez · CHI 2023 – ACM Conference on Human Factors in Computing Systems · 🏆 Best Paper Award

📄 Paper (arXiv) 💻 Code (GitHub) 🔗 ACM DL 📋 BibTeX

[ Demo video ]

Abstract. Designing effective data visualizations typically requires expert knowledge and iterative refinement. We present VizRL, a system that uses reinforcement learning from human feedback (RLHF) to automatically adapt visualization layouts and encodings to individual users' analytical tasks. Through a Wizard-of-Oz study and a fully automated deployment, VizRL reduces the time to insight by 35% compared to static, expert-designed dashboards, with particularly strong gains for users without prior visualization training.

Problem

Data visualization design is hard. Choosing the right chart type, encoding, and layout for a given analytical question typically requires expertise that most users lack. Automatic recommendation systems either rely on rigid rule sets or require large amounts of labeled training data specific to each dataset.

We ask: can a system learn good visualization policies on the fly, driven entirely by real-time human feedback, without requiring pre-labeled training corpora?

VizRL System

VizRL frames visualization adaptation as a sequential decision problem. At each step:

  1. The system observes the current visualization state and inferred user query.
  2. A policy network proposes modifications (chart type, color encoding, axis scale).
  3. The user implicitly signals satisfaction through interaction patterns (dwell time, zoom actions, export events) or explicit thumbs-up/down feedback.
  4. A reward model translates these signals into a scalar reward used to update the policy.

Key Findings

  • 35% faster time-to-insight vs. static dashboards.
  • 28% improvement for novice users; 15% for experts.
  • Implicit feedback signals were sufficient — explicit feedback was not required.
  • Policies generalized across datasets after as few as 10 interactions.
