Adaptive Neural Interfaces for Human-in-the-Loop Machine Learning

Shakeel Hassan, Jane Doe, John Smith · NeurIPS 2024 – Advances in Neural Information Processing Systems

📄 Paper (arXiv) 💻 Code (GitHub) 📋 BibTeX

[ Teaser figure / demo video ]

Abstract. We present Adaptive Neural Interfaces (ANI), a framework that dynamically adjusts the complexity of machine learning model explanations based on real-time signals from human users. By combining attention mechanisms with lightweight online learning, ANI reduces cognitive load by up to 40% while maintaining task accuracy above baseline levels. We validate ANI through a series of controlled user studies with 120 participants and demonstrate its effectiveness on three benchmark datasets spanning vision, language, and tabular domains.

Motivation

Human-in-the-loop (HITL) machine learning systems ask users to interact with, validate, or correct model predictions. Yet most systems present a one-size-fits-all interface that ignores user expertise, fatigue, and cognitive context. Experts are overwhelmed with unnecessary detail; novices are underserved by opaque summaries.

ANI addresses this gap by treating the interface itself as a learnable component, updated in real time based on implicit signals (response latency, correction rate) and explicit preferences.

Method

Our approach consists of three modules:

  • Signal Encoder: encodes behavioral signals into a low-dimensional user state vector using a lightweight LSTM.
  • Interface Controller: maps the user state to interface parameters (detail level, layout, highlighting priority) via a small policy network.
  • Reward Model: uses correction rate and self-reported confidence to generate rewards for online RL updates.

Results

Across three domains, ANI achieves:

  • 40% reduction in average decision time vs. static interfaces.
  • 12% improvement in annotation accuracy for novice users.
  • No statistically significant degradation for expert users.
